이 책에서의 실습을 위해 vscode extension을 몇가지 사용

1. 프리티어 (코드 정렬)
2. 테일윈드 css (스타일링)
3. 헤드윈드 (테일윈드 css 클래스 분류기)
4. POSTCSS (CSS 구문 강조)

-------------------------------------------------------

컴포넌트 스타일링을 위해 tailwind css라는 프레임워크를 사용한다.
=> Tailwind css intellisense extension 설치

헤드윈드는 테일윈드 css관련 코드를 이해하기 쉬운 순서로
재배치해주는 extension

PostCSS Language Support
=> @tailwind bae; 같은 비표준 CSS구문을 지원

타입스크립트 설정

아래의 명령어로 package 설치
> npm i -g typescript ts-node

아래의 명령어로 package 버전 확인 가능
> tsc -v
> ts-node -v
