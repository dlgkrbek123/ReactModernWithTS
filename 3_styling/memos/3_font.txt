css는 길이를 표현할때 px단위를 사용한다. px은 직관적이지만 
"몇글자정도의 길이" 같은 텍스트의 표시길이를 표현할떄는 적합하지 않다.

px말고도 영문자 "M"을 의미하는 em과 "root M"을 의미하는 rem이라는 단위를 제공
=> rem은 html, body애 설정된 기본 글꼴을 기준으로 'M'문자의 높이를 의미한다.
=> 1rem은 'M'문자 1개 높이

CSS는 텍스트의 글자 크기를 지정할 수 있게 font-size, line-height 속성을 제공
=> font-size : 글꼴의 크기, 글자의 높이
=> line-height: 글자의 높이, font-size에 여분의 높이를 더한 값

각 줄마다 line-height만큼의 높이를 가지고, 각 줄의 텍스트는 line-height보다 
조금작은 font-size가져 행간이 조금씩 떨어져 보이게 해준다.

브라우저는 h1~h6, p요소에 각기 다른 font-size와 line-height의 기본값을 설정한다.
브라우저마다 조금씩 달라서 결과물이 다르게 노출될 수 있다.
tailwind는 이 값을 reset하고 글자크기에 관한 클래스를 제공한다.

text-{size}
=> xs, sm, base, lg, xl, 2xl, 3xl, 4xl, 5xl
=> font-size, line-height를 정해준다.

--------------------------------------------------------------------------------

tailwind는 font-weight를 설정해주는 class를 제공해준다.

font-thin: 100
font-light: 300
font-normal: 400
font-medium: 500
font-bold: 700
font-black: 900

줄바꿈 문자 설정가능하다
=> 기본적으로 개행문자는 HTML에서 공백으로 간주되어 무시된다.
=> white-space 스타일 속성값으로 \n과 같은 공백에 대한 해석을 설정
=> tailwind에서 이를 위한 클래스를 제공한다.

whiespace-{value}
