--------------------------------------------------------------------------------

Daisyui는 class를 제공해줘서 이를 적용만 하면 UI를 스타일링 할 수 있다.
이를 CSS 컴포넌트라고 부른다.

--------------------------------------------------------------------------------

컬러 테마는 중요하다
=> 가장 많이 사용되는 색상은 primary
=> 두번째로 많이 사용되는 색상은 secondary
=> 이외에도 accent, info같은 클래스도 지원한다.

--------------------------------------------------------------------------------

버튼 CSS 컴포넌트는 사이즈와 색상을 class로 설정가능하다.
