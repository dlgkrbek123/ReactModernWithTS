---------------------------------------------------------------------------------

> npm i redux react-redux @reduxjs/toolkit

=> 메타는 리액트 발표시 Flux 아키텍쳐도 같이 발표했음
=> Global State를 관리

---------------------------------------------------------------------------------

리덕스는 Context와 비슷하게 사용가능하다
=> Provider와 hooks
=> store는 Global State가 저장된 저장소
=> reducer는 새로운 state를 리턴하는 함수
      (state, action) => newState
=> action은 type이라는 속성을 가지는 객체

RTK는 configureStore라는 함수를 제공

---------------------------------------------------------------------------------

useSelector로 컴포넌트에서 store에 접근가능
useDispatch로 dispatch 함수를 사용가능

/* ReduxClock */

---------------------------------------------------------------------------------

useReducer는 리덕스의 reducer와 똑같은 기능을 수행가능
=> useState, useCallback을 간결하게 구현가능
=> 컨텍스트없이 사용한다.
=> 호출한 컴포넌트 내에서만 유효한 지역상태

---------------------------------------------------------------------------------

combineReducers로 여러개의 reducer를 합친다.
=> action.type을 아래와 같이 명명한다
=> 액션은 모든 리듀서에 전달된다
=> action.type이 겹치지 않게 명명 + 가독성
  
  @{name}/{action}


=> 리듀서는 순수함수여야한다.
=> store의 비교는 !== 로 되므로 변경되었을 경우 깊은 복사를 해야한다.
